# Fake Casino Club

A fun browser-based gambling website with fake money, daily streak rewards, and fourteen mini games.

## Features

- Daily bonus starting at $100
- Streak bonus increases after 3 consecutive days:
  - day 1-3 = $100
  - day 4 = $200
  - day 5 = $300
- Games included:
  - Mines (larger board and numbered tiles)
  - Towers
  - Crash
  - Plinko
  - Dice (visual dice faces shown)
  - Blackjack with card visuals
  - Coin Flip
  - Spin Wheel with spinning animation
  - Slots
  - Roulette
  - Scratch Card
  - Keno
  - Baccarat
  - Treasure Hunt
- Login and profile with avatar selection
- Local leaderboard showing the top 10 players by balance
- Balance and history stored in browser local storage

## How to use

1. Open `index.html` in your browser.
2. Claim your daily bonus using the button in the top card.
3. Choose a game and place a fake bet.
4. Your balance and history are saved locally in your browser.
